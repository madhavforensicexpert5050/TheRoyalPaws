Premium pet store, grooming, and dog boarding website for **The Royal Paws**, Bathinda — built with Next.js (App Router), React, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Production Build

```bash
npm run build
npm run start
```

## Tech Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- Static generation (SSG) for all routes

## Project Structure

- `app/` — routes (main pages, service pages, location pages, legal pages)
- `components/` — shared UI components (Header, Footer, Hero, cards, layouts, schema helpers)
- `data/` — site configuration, navigation, services, locations, FAQs, and JSON-LD schema builders
- `public/images/` — logo, favicon assets, and brand photography
